¿Qué son la animaciones?

  • La transición que ocurre entre un punto A y un punto B, animación es generar una sensación de movimiento .
  • Son parte natural de las interfaces web.
  • Son una forma natural de enseñanza para el usuario. Hay acciones que tienen un tiempo de espera muy largo y otros que no, es allí cuando se pueden usar las animaciones para mostrarle al usuario que algo paso, y no sea algo que sale por arte de magia.
  • Ayudan a conectar una interfaz que solo está programada en algo que realmente está vivo.
  • La acción de un usuario interactuando con la interfaz y que ésta le de feedback de lo que está ocurriendo.

Nota: Las animaciones en una interfaz web son necesarias, ya que ninguna acción que ocurre en ésta se da de forma instantánea.

Transiciones

Es una propiedad de css3, que se utiliza con la propiedad transition, Para que una transición ocurra debe existir una interacción del usuario contemplada en un sitio web. Las transiciones CSS pueden especificarse usando varias propiedades:

  • property: Se le especifica que propiedades van a cambiar en la transición. 
                    transition-property: width, height;
  • duration: Cuánto tiempo va a durar la transición. 
                    transition-duration: 1s;
  • delay: Cuánto tiempo va a tardar la transición en ocurrir (puede ser opcional). 
                    transition-delay: 250ms;
  • timing-function: Qué tan suave puede ser la animación, de qué forma se calculan los valores intermedios de la transición.
    Tambien es opcional y su valor por defecto es ease. 
                    transition-timing-function: ease;
  • Se puede colocar las propiedades en una sola línea, cada propiedad a cambiar se separa por una coma. 
                    transition: width 1s 250ms ease, height 1s 250ms ease;

Pasar mouse por el circulo, para ver la transición.

Transiciones multiples

Si queremos animar varios elementos, la interacción debe ser con su contenedor padre, a ese contenedor se le coloca el hover u otra forma de interacción con el usuario.

Ahora para que la transición ocurra en los dos elementos se debe hacer hover (pasar mouse por encima) al contenedor padre y no a los elementos.

Código:

              Tiempo de retardo entre el momento en que el elemento se carga y el comienzo de la secuencia de la animación.
              .animation-content{
                position: relative;
              }

              /*Circulos hijos*/
              .animation-content .circle{
                position: absolute;
                height: 70px;
                width:70px;
                transition : 1s;
              }

              .animation-content .circle.a{
                left: 0;
              }

              .animation-content .circle.b{
                right: 0;
              }

              /*Interacción se agrega al contenedor padre*/
              .animation-content:hover .circle.a{
                left: calc(50% - 90px); /*se calcula la cantidad a la que se debe move con calc*/
              }

              .animation-content:hover .circle.b{
                right: calc(50% - 90px);
              }

Nota:
Para que la transición funcione debe por lo menos tener especificado la propiedad duration. Si la propiedad transition-property no se declara, quiere decir que los cambios van a aplicarse a todas las propiedades del elemento y timing-function viene por defecto en ease.

Transformaciones

Algunas de las cosas que podemos lograr en un elemento con transform son:

  • Rotar un elemento. rotate, se mide en grados (deg).
  • Sesgar un elemento. skew, también se mide en grados.
  • Cambiar la posición de un elemento. translate(pixeles), el elemento puede moverse en x, y y z.
  • Cambiar el tamaño de un elemento. scale(1 = 100%, 2 = 200%, .5 = 50%).
            transform: rotar | sesgar | posición | tamaño;
          
            transform: rotate(5deg)  skew(10deg) translate(100px) scale(1.5);

Transformaciones de rotación

Se puede rotar un elemento de muchas formas con animaciones.

  • rotate: Define una rotación 2D, con el ángulo que se especifica en el parámetro. 
                    transform: rotate(45deg);
  • rotateX: Define una rotación 3D a lo largo del eje X. 
                    transform: rotateX(10deg);
  • rotateY: Define una rotación 3D a lo largo del eje Y. 
                    transform: rotateY(10deg);
  • rotateZ: Define una rotación 3D a lo largo del eje Z. 
                    transform: rotateZ(10deg);
  • Combinación. 
                    transform: rotateX(10deg) rotateY(45deg) rotateZ(25deg) ;
  • rotate3d: Define una transformación a escala 3D. Se coloca 1 en los grados que se desea girar. 
                    transform: rotate3d(x,y,z, rotación);
                transform: rotate3d(1,0,0, 45deg);

Nota:
La rotación se da al sentido de las manecillas del reloj, para cambiar la dirección de esa rotación se pueden usar grados negativos.

rotateX(360deg)
rotateY(180deg)

Transformaciones de traslación y perspectiva

Translate nos sirve para poder mover un elemento arriba, abajo, derecha e izquierda. Utilizamos la propiedad perspective para poder darle profundidad a un elemento y obtener un efecto 3D.

  • translate: Define una translación 2D. 
                    transform: translate(x);
                transform: translate(100px);
                transform: translate(x, y);
                transform: translate(100px, -50px);
  • translateX: Define una translación (desplazamiento), usando solo el valor para el eje X. 
                    transform: translateX(100px);
  • translateY: Define una translación (desplazamiento), usando solo el valor para el eje Y. 
                    transform: translateY(100px);

Utilizamos la propiedad perspective para poder darle profundidad a un elemento y obtener un efecto 3D. Para ver los efectos en el eje Z con perspective utilizamos perspective-origin podemos definir el punto de origen de la perspectiva, podemos también combinaciones:

  • top right
  • top left
  • bottom left
  • bottom right
Al contenedor padre, del objeto que queremos agregarle profundidad: 
              body, .container-father {
                perspective: 200px; 
              }

Siguiendo con las translate:

  • translateZ: Tambien podemos movernos en el eje Z, para dar profundidad. 
                    transform: translateZ(100px);
  • translate3d: Define una translación a escala 3D. 
                    transform: translate3d(x,y,z);
                transform: translate3d(100px, 100px, 100px);

Transformaciones de escala

Transform-scale nos permite modificar el tamaño de nuestros elementos a nuestro gusto.
Esta propiedad solo recibe valores numéricos, y dichos valores representan porcentajes, por ejemplo:

  • 0 = 0% del tamaño
  • .5 = 50% del tamaño
  • 1 = 100% del tamaño
  • 1.5 = 150% del tamaño
  • 2 = 200% del tamaño
  • scale(valor): De esta manera escalamos el elemento tanto en X como Y por igual. 
                    transform: scale(1.5);
  • scale(valorX, valorY): De esta manera escalamos tanto en X, como Y pero con diferentes valores. 
                    transform: scale(.8, 2);

Transformaciones de sesgados

Sesgar es esa inclinación que puede sufrir un elemento gracias a una transformación en CSS.
Se puede sesgar un elemento tanto de un solo eje, como de ambos:

  • skew: Sesgar en el eje X. 
                    transform: skew(20deg);
  • skew(valorX, valorY): Sesgar en el eje X e Y. 
                    transform: skew(10deg, 45deg);
  • Sesgar en X e Y, de manera independiente: 
                    transform: skewX(10deg);
                transform: skewY(45deg);

Nota:
Esta propiedad recibe valores en grados (deg) y radianes (rad).

Punto de transformación

El punto de transformación de un elemento está clavado en alguna parte de la interfaz, y por defecto se encuentra en el centro.

Sin embargo, podemos jugar con la posición de este punto según el tipo de animación que queramos crear.

            transform: rotate(45deg);
            transform-origin: bottom left;

Animaciones

Configurando animación: Se utiliza la palabra clave @keyframes seguido del nombre que queramos darle a la animación, dentro de las llaves se especifica lo que hace dicha animación y esto se hace con estados:

            /* @keyframes nombreAnimación */
            @keyframes cuadrado{
              /* Estado inicial de animación */
              0%{
                opacity:1;
              }
              /* Estado final de animación */
              100%{
                opacity:0;
              }
            }

Para crear una animación CSS usaremos la propiedad animation y sus sub-propiedades, con las cuales podemos configurar el ritmo, la duración, etc.

Las propiedades de animation son:

  • animation-name: Especifica el nombre de la regla @keyframes que describe los fotogramas de la animación. 
                    animation-name: rebote;
  • animation-duration: Indica la cantidad de tiempo que la animación consume en completar su ciclo (duración). 
                    animation-duration: 1s;
  • animation-delay: Tiempo de retardo entre el momento en que el elemento se carga y el comienzo de la secuencia de la animación. 
                    animation-delay: 1s;
  • animation-direction: Indica si la animación debe retroceder hasta el fotograma de inicio al finalizar la secuencia o si debe comenzar desde el principio al llegar al final. 
                    animation-direction: reverse;/*Empieza desde el último estado 100%.*/
                animation-direction: alternate; /*La animación al terminar empieza nuevamente pero desde el último estado en vez del inicio.*/
  • animation-iteration-count: El número de veces que se repite. Podemos indicar infinite para repetir la animación indefinidamente. 
                    /*Cuantas veces va a ocurrir la animación*/
                animation-iteration-count: 1;
                animation-iteration-count: infinite;
  • animation-play-state: Permite pausar y reanudar la secuencia de la animación. 
                    animation-play-state: running; /*Valor por defecto*/
                animation-play-state: paused;
  • animation-timing-function: Indica el ritmo de la animación, es decir, como se muestran los fotogramas de la animación, estableciendo curvas de aceleración. 
                    animation-timing-function: ease; /*Valor por defecto*/
                animation-timing-function: ease-in; 
                animation-timing-function: ease-in-out; 
                animation-timing-function: linear; 
                animation-timing-function: cubic-bezier(1, 1, 1, 1);
  • animation-fill-mode: Especifica qué valores tendrán las propiedades después de finalizar la animación (los de antes de ejecutarla, los del último fotograma de la animación o ambos). 
                    animation-fill-mode: forwards;

Aceleración y curva de bezier

Aceleración y movimientos curvos a los elementos web por medio de animation-timing-function.

Valores de animation-timing-function:

  • ease: Por defecto.
  • ease-in: Se suaviza al entrar o al inicio.
  • ease-out: Se suaviza al salir o al finalizar.
  • ease-in-out: Se suaviza al entrar y salir, básicamente se acelera solo la mitad de la animación.
  • steps(frames por segundo): como el lag en los juegos en linea, mientras más bajo será menos fluido, entre más fps más suave la animación.

Múltiples animaciones

Para nombrar más de una animación podemos nombrarlas en la misma línea de código:

            animation-name: nombre1, nombre2;

Animaciones CSS desde JS

Para detectar eventos de animaciones a través de Javascript se pueden usar los siguientes:

  • animationstart: Para ver si la animación inicio.
  • animationend: Para ver si la animación termino.

Tambien se puede crear animaciones desde Javascript, ejemplo:

            $cuadrado.style.animationName = 'cuadrado';
            $cuadrado.style.animationDuration = '1s';

Para crear animaciones desde JavaScript se utiliza element.animate, que es parte de la especificación del API de animaciones de JavaScript.

            // element.animate(keyframes = [], option = {})
            $pelota.animate([
              // from
              transform: 'translateX(0)'
            },
            // to
            {
              transform: 'translateX(500px)'
            }  
            ], {
              duration: 1000,
              delay: 1000,
              direction: 'normal',
              easing: 'linear',
              iterations: Infinite,
              fill: 'forwards'
            })
