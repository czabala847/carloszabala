Centrar contenido

Para hacer un contenedor general y centrarlo se puede usar: 

            grid-template-columns: minmax(auto, 1024px);
            justify-content:: center;

Grid area

Si no queremos que una sección ocupe todo el ancho o todas las columnas del grid, se pueden colocar un punto(.) en la propiedad grid-template-areas:

            grid-template-areas: "icon-area title-area-con"
                                "content-area-con content-area-con"
                                "time time"
                                "form-area-con form-area-con";

Curso de CSS Grid Layout

El módulo CSS Grid Layout nos permite generar distribuciones del contenido en nuestros sitios web de forma eficiente. Controla esta estructura bidimensional de filas y columnas fácilmente con propiedades de CSS logrando un mejor flujo que el que se tiene al utilizando otras propiedades de CSS
