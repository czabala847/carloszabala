Logo animado

Css: 

              .animation-logo-img .logo-c {
                animation-name: logo-c-in;
                animation-duration : 2s;
                animation-iteration-count : 1;
                animation-direction: alternate;
              }

              .animation-logo-img .logo-z {
                opacity : 0;
              }
              
              .animation-logo-img .logo-z.active-animation {
                animation-name: logo-z;
                animation-duration: 1s;
                animation-iteration-count: 1;
                animation-fill-mode: forwards;
              }

              /*Configurando animación*/

              @keyframes logo-c-in{
                0% {
                  opacity: 0;
                  transform: translateX(150px) scale(.4) rotate(360deg);
                }
                50% {
                  transform: translateX(150px) scale(1) rotate(0);
                }
                75% {
                  opacity: 1;
                  transform: translateX(0px) scale(1) rotate(0);
                }
                100% {
                  opacity: 1;
                  transform: translateX(0) scale(1) rotate(0);
                }
              }

              @keyframes logo-z {
                0%
                  opacity: 0;
                  transform: translateX(-150px);
                }
                80% {
                  opacity: .8;
                  transform: translateX(50px) rotate(8deg);
                }
                100% {
                  opacity: 1;
                  transform: translateX(0) rotate(0);
                }
              }

Javascript: 

              const $logoC = document.getElementById('logo-c');
              const $logoZ = document.getElementById('logo-z');

              $logoC.addEventListener('animationend', () => {
                $logoZ.classList.add('active-animation');
              });

Javascript: 

              const $btnModal = document.getElementById('btn-modal');
              const $modal = document.getElementById('modal');
              const $closeModal = document.getElementById('close-modal');
              const $overlay = document.getElementById('overlay');

              $btnModal.addEventListener('click', () => {
                $overlay.style.opacity = '1';
                $overlay.classList.remove('out');
                $modal.classList.remove('out');
                $modal.classList.add('active');
              })
            
              $closeModal.addEventListener('click', () => {
                $modal.classList.add('out');
                $modal.classList.remove('active');
                $overlay.classList.add('out');
              });

Javascript: 

              const $btnNotification = document.getElementById('btn-notification');
              const $iconBell = document.getElementById('icon-bell');

              $btnNotification.addEventListener('click', () => {
                const count = Number($iconBell.getAttribute('data-count')) || 0;
                $iconBell.setAttribute('data-count', count + 1);
                $iconBell.classList.add('show-count');
                $iconBell.classList.add('active');
              });


              $iconBell.addEventListener('animationend', () => {
                console.log('hi');

                $iconBell.classList.remove('active');
              })

Sprite

Propiedad utilizada: animation-time-function: steps(). Para hacer que la animación suceda en pasos.
Sprites: Conjunto de imágenes.

Los pasos steps estan relacionados con el eje x del background-positions Para calcular cuantos pixeles nos debemos estar moviendo hay que dividir el tamaño total del Sprite de la imagen por el número de imágenes que tenga.
Buscar imagenes sprites:

Sprite database Spriters resource

html: 

                  <div  class="item-sprite animation-like">
                    <div  id="like" class="like"></div >
                  </div >

css: 

                  .animation-like{
                    display: flex;
                    justify-content: center;
                    align-items: center;
                  }

                  .animation-like .like{
                    background-image: url('../img/like.png');
                    /*reducir a la mitad el tamaño del sprite*/
                    background-size: 1450px 50px; 
                    height: 50px;
                    width: 50px;
                    cursor: pointer;
                  }

                  /*Activar la animación*/
                  .animation-like .like.isliked{
                    animation: like 1s forwards steps(28);
                  }

                  /*Configurar la animación*/
                  @keyframes like{
                    0%{
                      background-position: 0;
                    }
                    100%{
                      /*Ir moviendo hacia la derecha la imagen*/
                      background-position: right;
                    }
                  }

Javascript: 

                  const $like = document.getElementById('like');

                  like.addEventListener('click', () => {
                    $like.classList.toggle('isliked');
                  });
