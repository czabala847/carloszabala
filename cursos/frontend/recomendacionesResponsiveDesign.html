Media Querias más usados

  • 1024px
  • 768px
  • 480px
  • 320px

  • Móviles: entre 320 y 480 píxeles.
  • Tablets: entre 768 y 1024 píxeles.
  • Pantallas grandes: más de 1024 píxeles o más.

Max-width

El max-width, depende de sus hijos, si le colocamos a un contenedor un tamaño máximo de 1000px pero sus hijos no llegan a cumplir o sobre pasar ese ancho, el contenedor no va a tener los 1000px de ancho, esto causa que los hijos de ese contenedor se alineen al centro.

Un truco para corregir esto es colocarle un tamaño de 100%, del tamaño máximo que debe tener.

            .container{
              max-width: 1000px;
              width: 100%; /*truco del ancho del 100%*/
              margin: 0 auto; /*Centrar contenedor*/
              display: inherit;
              justify-content: inherit;
              align-items: inherit;
              height: inherit;
              flex-wrap: inherit;
            }

Esto también pasa cuando esta activo el display flex, otra forma alternativa del width del 100% es usar propiedades del flex. Esto quiere decir que el elemento toma todo el ancho que tiene disponible El problema del width 100%, es que si le agregamos un padding, el ancho va a ser mayor al 100%.

              flex:1;
