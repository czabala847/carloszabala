Forma correcta de poner negritas (strong, b, font-weight)

Para resaltar una palabra en negrillas, en html5 existen varias posibilidades, pero hay una serie de recomendaciones a tener en cuenta, para asi apostar por buenas practicas en el desarollo web y tener una mejor semantica del lenguaje.

  • Si solo queremos que aparezca en negrita, sin dotar de significado de importancia especial al texto, deberías envolverlo en algún elemento neutro (como <span>) y aplicarle la propiedad font-weight.
  • Si además realmente la frase o contenido en cuestión tiene especial importancia en un contenido, el uso de <strong> le da un enfoque semántico e indicará a los buscadores este hecho.
  • La etiqueta <b> se usa muy poco en la actualidad. Ocupa menos que la anterior y está reconocida por el estándar y por los navegadores, pero es un vestigio del pasado y realmente no la necesitas. Sería mejor usar la anterior si quieres darle el significado de importante. Salvo que requieras denotar varios niveles de importancia, como hemos visto más arriba.

Centrado de elementos

Se puede centrar un elemento html colocándole el valor de margin: 0 auto, cuando dicho elemento tiene display block.

Existen también propiedades equivalentes que nos permite alinear textos dentro de un contenedor, estas propiedades son: text-align y vertical-align.

Imagenes

Para que las imágenes se distribuyan de una mejor manera, cuando tengan un ancho y alto fijo, se utiliza la propiedad:

object-fit: cover

Módelo de cajas

Overflow

Si nuestro contenido rebasa o es mucho más grande que nuestras cajas (contenedores) se puede usar la propiedad overflow para generar un tipo de scroll dentro de las cajas.

  • hidden: Esta propiedad hace que cuando los elementos se desborden no se muestre la parte que se desborda, solo se muestra en pantalla lo que este dentro del alto y ancho que posee la caja.
