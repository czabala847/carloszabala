Instalación

Descargar tanto el jdk y jre en sus más resientes versiones, de pueden bajar desde la página de Oracle.

Configurar entorno de desarrollo

En la consola de nuestro sistema operativo (CMD) podemos ejecutar los siguientes comandos una vez tengamos java instalado:

  • java -v o java-version: Para ver la versión de java que tenemos instalada.
  • javac: Ver el resto de opciones que tenemos una vez instalado java.
Solucionar error javac

Identificando objetos

Los nombres y atributos de los objetos por lo general son sustantivos.
user, session
La operaciones suelen ser verbos o sustantivo y verbo.
login, makeReport

Es recomendado que al analizar una clase empecemos "desglosando" los objetos en papel, escribiendo los atributos que puedan tener estos objetos, para así tener una lluvia de ideas.

Modificadores de acceso

Es recomendado usar la encapsulación, mediante modificadores de acceso getters y setters, ver más en: Modificadores

Nota: cuando empieces a copiar y pegar código en Java, quiere decir siempre que hay una mejor forma de hacerlo, utilizando las buenas prácticas.

Herencia

Cuando encontramos redundancia, es decir, cuando vemos que ciertos atributos y métodos de algunas clases son los mismo, es cuando debemos aplicar la herencia Reutilizar código, para escribir lo menos posible. Más sobre herencia.
